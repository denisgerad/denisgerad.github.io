Hands-On LLM – Foundations to Agentic Systems

A structured, engineering-first learning repository designed to demystify how Large Language Models actually work — from tokenization and embeddings to Retrieval-Augmented Generation (RAG), code generation, and agentic reasoning.

Part 1 focuses on core mechanics such as tokenization flow, chunking strategies, vector search, extractive vs generative summarization, and causal code generation. The project separates retrieval logic from generation to clearly illustrate how LLM systems are architected.

Part 2 evolves into lightweight agentic systems, demonstrating ReAct-style reasoning, tool invocation, memory loops, multi-step planning, and evaluation workflows — without relying on heavy abstraction frameworks.

Tech: Python, Transformers, FAISS/Chroma, Ollama (Optional), Mistral, ReAct Loop, RAG Architecture, Agent Design Patterns

Crew-AI (HR Recruiting Prototype)

An AI-powered HR recruiting prototype designed to evaluate how multiple LLM agents can collaborate on complex tasks such as CV parsing, candidate scoring, and decision-making.

The project explores agent orchestration, role-based reasoning, and task delegation using CrewAI, while running models locally via Ollama. The focus is on understanding agent workflows and decision boundaries rather than building a production ATS.

Tech: Python, CrewAI, Ollama, Multi-Agent LLMs

GitHub Repo

AI Agent Prototype

A research-oriented AI agent framework built to study agentic reasoning, conversation memory, and dynamic tool selection using locally hosted LLMs. The agent can reason over PDFs (RAG), web content, APIs, and conversation history.

A key extension of this project is an investigation-first Debug Agent that avoids blind code refactoring. Instead, it detects environmental signals (e.g. “works on PC but not mobile”), asks clarifying questions, generates verification steps, and proposes multiple fix strategies only after validation.

Tech: Python, LangChain, Ollama (Mistral), ChromaDB, Tool Calling, Agent Memory

GitHub Repo



TranslateGemma – Multilingual Translation

A local-first benchmarking and demo pipeline built around Google’s TranslateGemma model, designed to evaluate difficult literary English translations into Indian languages.

The project performs batch translation using Ollama, applies automatic quality scoring.

Tech: Python, Ollama, TranslateGemma, NLP Evaluation, TTS (ElevenLabs & Coqui), ffmpeg, Canva

Mini Cloud Platform

A hands-on, production-style cloud-native learning platform built locally using Docker Desktop, Kubernetes, Terraform, and LocalStack. The project simulates real-world cloud architecture and workflows without requiring an actual AWS account.

The platform hosts a full-stack application with a React frontend, backend API, and PostgreSQL database, exposed via a Kubernetes Ingress that functions like an AWS Application Load Balancer. Infrastructure and application layers are fully decoupled using Infrastructure as Code and runtime configuration.

Tech: Kubernetes, Docker, Terraform, LocalStack (S3, SQS), React, FastAPI/Django, PostgreSQL, NGINX Ingress, Bash, Linux, Git

GitHub Repository


MERN Messenger Starter

A real-time chat application built as a hands-on exploration of the MERN stack, focusing on authentication flows, token handling, and WebSocket-based messaging.

The project explores how modern web applications handle live messaging, user presence, and session-based authentication, with a UI inspired by WhatsApp’s dark theme.
